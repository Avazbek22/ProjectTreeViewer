## 1) Бұл не және не үшін
ProjectTreeViewer жоба контекстін тез дайындайды: қалта/файл құрылымын көрсетеді, элементтерді белгілеуге және ағашты және/немесе файл мазмұнын алмасу буферіне көшіруге мүмкіндік береді. Code review, қолдау, оқыту және жоба құрылымын AI‑көмекшіге беру үшін ыңғайлы.

Маңызды: қолданба тек оқу режимінде жұмыс істейді және файлдарды өзгертпейді.

## 2) Жоба ағашы
* Файл/қалта түрі иконкалары.
* Түйіндердегі чекбокстар.
* Ашып/жабу.

Чекбокстар: түйінді белгілеу оның ұрпақтарына қолданылады; ата‑ананың күйі автоматты қайта есептеледі.

Ыстық пернелер:
* Ctrl+E — бәрін ашу
* Ctrl+W — бәрін жабу (түбір ашық қалады)

## 3) File (Файл)
* Open folder… — түбір қалтаны ашу (Ctrl+O)
* Refresh — ағашты қайта оқу және қайта құру (F5)

## 4) Copy (Көшіру)
### Copy full tree
Жобаның толық ASCII‑ағашын көшіреді. Ctrl+Shift+C.

### Copy selected tree
Белгіленген түйіндер бойынша ағашты көшіреді. Таңдау болмаса — хабарлама шығады. Ctrl+Alt+C.

### Copy selected content
Тек таңдалған мәтіндік файлдардың мазмұнын көшіреді:
* файл болуы керек;
* бинарлық файлдар өткізіледі (null‑байт тексеруі);
* шығару сұрыпталып, бос жолдармен бөлінеді.

Ctrl+Alt+V.

### Copy tree + content
Ағаш пен мазмұнды біріктіреді:
* ағаш таңдау бойынша құрылады; қате болса — толық ағаш алынады;
* мазмұн тек мәтіндік файлдар үшін қосылады;
* мазмұн болмаса — тек ағаш көшіру.

Ctrl+Shift+V.

## 5) View (Көрініс)
### Масштаб (Zoom)
* Ctrl++ / Ctrl+= — үлкейту
* Ctrl+- — кішірейту
* Ctrl+0 — қалпына келтіру
* Ctrl + тінтуір дөңгелегі — бірқалыпты зум

Диапазон: 6–28, қалпына келтіру — 12.

### Compact Mode
Интерфейсті тығыздайды: ағашта және баптауларда аралықтар азаяды.

## 6) Search (Ағаш бойынша іздеу)
Іздеу жолағын ашады, сәйкестіктерді белгілеп, навигация береді.
* Ctrl+F — көрсету/жасыру
* Enter — келесі нәтиже
* Shift+Enter — алдыңғы
* Esc — жабу

Іздеу тек керек бұтақтарды ашады; сұраныс тазаланса, ағаш жиналады (түбірден басқа).

## 7) Filter (Атау бойынша)
Сүзгіге сай түйіндер мен олардың ата‑аналарын көрсетеді. Жаңарту кешіктіріледі (debounce).
* Ctrl+Shift+N — сүзгіні көрсету/жасыру
* Esc — сүзгіні жабу

## 8) Options (Баптаулар панелі)
Оң жақтағы тізімдер:
* Font (ағаш шрифті) — Apply түймесімен
* Ignore options
* Extensions
* Root folders

Әр тізімде All ауыстырғышы бар.

Ctrl+P — панельді көрсету/жасыру.

## 9) Ignore және Smart Ignore
### Қарапайым ережелер
Bin/obj, жасырын элементтер, нүктелі элементтер сияқты «шу» артефакттарын алып тастау.

### Smart Ignore
Қосымша «ақылды» ерекшеліктер (VCS/IDE қалталары және қоқыс файлдар). Frontend ережелері тек тиісті белгілер табылғанда қосылады (package/lock файлдар).

## 10) Ағаш қалай құрылады
Нәтижеге әсер етеді:
* таңдалған Root folders
* таңдалған Extensions
* Ignore rules
* белсенді Filter

Қолжетімсіз болса, түйін Access denied деп белгіленеді.

## 11) Иконкалар
Таңдау тәртібі:
1) түйін түрі (қалта/файл)
2) нақты атау (ереже болса)
3) кеңейтім
4) әйтпесе — unknown

## 12) Theme (Тақырып және эффекттер)
Тақырып popover‑і келесіні ауыстырады:
* ашық/қараңғы тақырып
* Transparent / Mica / Acrylic
* интенсивтілік, blur, контраст және шекара параметрлері

[Beta] белгісі бар тармақтар нұсқалар арасында өзгеруі мүмкін.

## 13) Help → About (Бағдарлама туралы)
Қолданба туралы ақпарат пен репозиторий сілтемесі бар popover ашады.
* Esc — popover жабу

## 14) Тілдер
Тіл Language мәзірі арқылы таңдалады.

## 15) Ескертпелер
* Әдепкіде қараңғы тақырып және мөлдірлік эффекттері қосулы (қолдау болса).
* Рұқсат жеткіліксіз болса, қолданба әкімші құқықтарымен қайта іске қосуды ұсынады; бас тартылса — хабарлама көрсетіледі.
